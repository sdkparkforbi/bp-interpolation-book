PART 2 (개정판). 120명의 시간별(1시간 간격) 인터폴 SBP·DBP를 데이터로 — 앞 5일로 뒤 2일을 예측한다. 모든 평가는 인터폴 그리드 기준(168h), 학습구간·테스트구간을 따로 본다. 위상(시간대)을 쓰는 모형을 총동원하고, static vs dynamic을 비교한다. 평가는 유의하면 충분.
1

예측의 구조 — 인터폴 그리드, 5일/2일

  1. 데이터 = 시간별 인터폴. 각 환자를 168h(7일×24h) 인터폴 곡선으로(각 날은 그날 측정값만 인과적으로). 7일 완전 119명.
  2. 5일 학습 / 2일 테스트. day0~4(120h) 학습 → day5·6(48h) 예측. SBP·DBP를 각각(Single) 또는 동시(Multi)로.
  3. 위상 반영. 모든 피처모델에 시간대 하모닉(sin/cos 24·12·8h) + 개인화(평균BP·군집·인구통계) + 래그.
  4. static vs dynamic. static=5일로 한 번에 2일 / dynamic=day5 관측 후 갱신해 day6.
  5. 평가. 전 구간 인터폴 그리드의 MAE — 학습구간(in-sample)과 테스트구간(forecast)을 각각 본다.
2

모델 동물원 + 리더보드

한 무대에 올린 14개 모형 — 기준선 전일복사 · 위상/계절 일평균프로파일·하모닉(스펙트럴)·위상커널·VAR(계절성)·VARX(외생) · ML Ridge·GBM·NN/MLP·Transformer (각 Single/Multi).

테스트구간 인터폴-MAE(dynamic), 119명. 금=최저. 색=그룹
✅ 핵심. SBP 최강 = GBM·M(4.40), DBP 최강 = 위상커널(3.53). 위상을 쓰는 모형·GBM·Transformer가 모두 천장(SBP≈4.4 / DBP≈3.53 인터폴)에 통계적으로 막상막하로 모인다. VARX·전일복사·MLP만 처짐.
3

학습 vs 테스트 — 과적합의 정체

왼=SBP, 오=DBP. 연두=학습구간(in-sample) / 빨강=테스트구간(forecast). 두 막대 격차가 클수록 과적합
🌀 결정적 관찰. VAR·VARX는 학습구간을 거의 완벽히 맞춘다(SBP MAE 0.71/0.73)—하지만 테스트구간에선 4.70/5.36로 폭증. 자기회귀가 짧은 개인 시계열의 학습구간을 외워버린 전형적 과적합. MLP도 학습 3.16 → 테스트 5.09로 벌어진다.

4

최적 래그

일평균프로파일: 평균에 쓰는 최근 학습일 수 L. L=5(가진 날 전부)에서 최저(금테)

최적 래그 = "가능한 많은 날". 더 많은 날을 평균할수록 위상 프로파일이 안정. L=1→3에서 급감 후 포화.

5

static vs dynamic

상위 7개 모델, 테스트 인터폴-MAE. 보라=static / 빨강=dynamic
dynamic이 위상·계절 모델에서 근소 우위 — day5를 관측해 프로파일을 갱신하면 day6가 더 정확(위상커널 4.48→4.41). ML은 차이 미미.
6

Single vs Multi

Single(자기 채널) vs Multi(SBP·DBP 교차채널). 테스트 인터폴-MAE
교차채널(Multi)은 일관된 이득이 없다 — GBM만 SBP에서 미세 우위(4.40 vs 4.43), 나머지는 무차. SBP·DBP가 강하게 동행해 서로의 정보가 중복.
7

예측 데모

샘플 환자 — 회색=인터폴 타깃(정답). 보라=static / 주황점선=dynamic. day5에선 동일, day6에서만 갈림

위상커널이 그 사람의 전형 곡선을 따라가며, day6는 day5 관측을 반영(dynamic)해 조정된다.

∎

결론

🏁 예측 최종(인터폴 기준). SBP는 GBM·M(4.40), DBP는 위상커널(3.53)이 테스트 최저. 위상 모형·GBM·Transformer가 천장에 막상막하. 최적 래그 L=5, dynamic 근소 우위.
핵심 교훈 둘. ① 학습구간 성능에 속지 말 것 — VAR·VARX·MLP는 학습은 완벽하나 테스트에서 과적합. ② 보간이든 예측이든, VAR·Transformer·스펙트럴까지 다 써봐도 혈압 내부 신호의 천장은 "그 사람의 위상(전형적 하루)". 천장을 크게 넘으려면 동시시점 외생관측(웨어러블)이 필요하나 짧은 개인 시계열에선 외생(VARX)이 과적합했다.
