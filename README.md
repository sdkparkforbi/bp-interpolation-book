# 혈압(BP) 보간 그림책 — Interpolation Picture Book

마스킹된 혈압(SBP/DBP)을 어떻게 가장 잘 복원하는가를 다룬 분석 보고서입니다.
가중규칙(단순평균/역시간/위상/합성) × 추정수준(개인별/군집별/전역)을 비교하고,
두 채널(SBP·DBP)을 함께 쓰는 경우까지 검증했습니다.

- 📖 보고서: `index.html` (GitHub Pages로 게시)
- 데이터: 120명 × 약 34세션, 평가 LOO masked-recovery MAE, 보간 전용(공변량 제외)

## 핵심 결론
- 시간·이웃·위상·두 채널 어떤 조합도 **개인평균이 천장**(차이 ~0.3 mmHg)
- 가장 나은 보간법: **합성커널 · 전역 추정**
- 동시시점은 SBP·DBP가 함께 마스킹되어, 두 채널 결합으로도 천장을 넘지 못함
