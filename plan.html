3. 데이터와 관찰값 구성

  • 120명 × 7일, 세션당 보통 2회 측정(1분 간격), 하루 ~5세션.
  • 관찰값 = session mean (연속 측정 Δt ≤ 3분을 한 세션으로, 전체 값 평균).
  • timestamp = 마지막 측정 시각 (워치 window (−∞, t+1분]이 세션 데이터를 모두 덮고 시점 정합).
  • Minimal editing: 명백한 artifact 2건만 제거(SBP=12, DBP=17). 저혈압 가능값(SBP=68 등)은 유지, 진짜 분포는 winsorize 안 함.
  • 결과: 4,169 session observations (표준 테이블 _feat_estimation_sess.pkl).
Measurement error (라벨의 noise component). 1분 반복측정으로 추정한 per-reading SD ≈ 4.8 (SBP) / 3.8 (DBP) mmHg (가정용 oscillometric, AAMI 기기 한계 수준). 첫 측정 +2.1 mmHg systematic bias(alerting)는 session mean으로 완화. within-person SBP variance의 약 18–36%가 measurement error, 나머지는 modellable 하지 않은 within-person physiology.
SBP 변동의 분해 — per-reading SD≈4.8, session mean이 within-person SD를 7.96→7.19로 낮추고 MAE 7.0→6.5. HR-BP는 정렬해도 r≈0.14.

4. 평가 원칙 (불가침)

1. Performance는 real measured points에서만 측정한다. interpolated value로 metric을 계산하지 않는다.
2. Interpolation은 자유로운 도구. feature 정렬, 중간 dense representation, 시각화, training augmentation에 써도 된다 — evaluation set만 실측으로 둔다.
3. Metric = MAE + BHS grade, train / valid / test 분리.
4. 안정성 = person-level resampling(≈100 splits), 비교는 paired Wilcoxon, 결과는 mean ± SD.

5. Study design (two-axis split)

Person split(cross-subject) × Time split(temporal). cross-subject 성분은 train people에서만 적합; test 사람의 자기 데이터는 calibration으로만, 평가점은 본인 future 실측점.

  • Main = deployment: 새 사람 × future days × real points.
  • Deployment performance curve: MAE vs #calibration days (0·1·…·6).
  • Axis-isolation diagnostics: person-only(순수 추정) vs time-only(순수 예측)로 difficulty attribution.

6. Target granularity (세 단위)

평균 단위가 커질수록 measurement-error variance가 1/n로 감소 → MAE 하락. 두 product: 임상용 daily average, 도전적 session momentary.

집계 단위별 — session/분당점 7.06(C) → 1일평균 4.55(A). 1일평균은 예측·실측이 대각선에 밀착(SBP r=0.89).

7. 핵심 결과 — 6일치로 7일을 예측 (통합 design, real points)

각 사람의 7일째를 비우고 1~6일치로만 적합 → 7일째의 session-point와 일평균을 예측. 사람 60:20:20, day7은 일주기·워치 적합에서 제외(누수 차단), 평가는 7일째 실측점에서만. 이것이 PART 1(보간)·2(예측)·3(추정)의 합류점.
